화면 설정 안내

Always On (화면 항상 켜기) 브라우저에서는 Wake Lock으로 화면 꺼짐을 막습니다. 실제 앱은 실행 중 자동으로 꺼짐 방지.
잠금화면으로 설정
1. 안드로이드 설정 › 잠금화면 › 위젯(또는 '항상 표시') 2. Desk Glance 위젯 추가 → 잠금 상태에서 오늘 일정 표시 (Phase 2 예정)
대기 화면(Standby) 위젯으로 설정
1. 홈 화면 길게 누르기 › 위젯 › Desk Glance 2. 원하는 크기(2×2/4×2) 배치 — 충전 대기화면은 'Always On'과 함께 사용 (Phase 2 예정)

캘린더 연결

Outlook 연결됨기기 계정으로 자동 연동 — 로그인 정보 저장 없음
1. Outlook 앱 설치 › 회사/개인 계정 로그인 2. Outlook 설정 › 계정 › '캘린더 동기화' 켜기 3. 기기 캘린더에 나타나면 이 앱에 자동 반영
Google Calendar기기 계정으로 자동 연동 — 로그인 정보 저장 없음
1. 안드로이드 설정 › 계정 › Google 계정 추가 2. 계정 동기화에서 '캘린더' 켜기 3. 기기 캘린더에 나타나면 이 앱에 자동 반영
네이버 캘린더CalDAV로 기기 캘린더에 연결
1. 네이버 설정에서 '캘린더 CalDAV' 사용 + 앱 비밀번호 발급 2. CalDAV 동기화 앱(DAVx⁵ 등)에 caldav.calendar.naver.com 계정 추가 3. 기기 캘린더에 나타나면 이 앱에 자동 반영
Apple iCloud공유 링크로 Google 캘린더에 구독
1. iCloud 캘린더 › 공유 › '공개 캘린더' 링크 복사 2. Google 캘린더 › 'URL로 추가'에 붙여넣기 3. Google 동기화를 타고 이 앱에 자동 반영
삼성 캘린더기기 캘린더와 같은 저장소 — 별도 연결 불필요
1. 삼성 캘린더의 일정은 기기 캘린더 저장소를 함께 씁니다 2. 삼성 계정만 로그인돼 있으면 이 앱에 자동 표시

각 연결에서 가져온 일정을 어느 구분(회사/개인)으로 표시할지 선택하세요. 색은 설정의 색상에서 정합니다. 안내는 안드로이드 기준.
